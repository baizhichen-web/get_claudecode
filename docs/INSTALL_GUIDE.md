# Claude Code 安装指南

## 快速安装

完整安装步骤请查看项目根目录的 [README.md](../README.md)。

如果你使用 Cursor / Trae 等 AI IDE，可以直接在 AI 对话框中输入：

```
帮我安装 Claude Code。请先克隆这个仓库：https://github.com/baizhichen-web/get_claudecode
然后按照项目中的文档一步步引导我完成安装。
```

AI 会自动完成所有步骤。

## 系统要求

- **操作系统**: Windows 10/11
- **PowerShell**: 5.0 或更高版本
- **Node.js**: 18.x 或更高版本（用于安装 Claude Code CLI）

## 手动安装

如果你需要手动安装，请参考 [README.md](../README.md) 中的「快速开始」章节，包含：
1. 下载 CLIProxyAPI
2. 创建配置文件
3. Qwen 登录授权
4. 配置 Claude Code
5. 启动服务
6. 验证连接
7. 运行 Claude Code

## 故障排除

请参考 [TROUBLESHOOTING.md](TROUBLESHOOTING.md)。
